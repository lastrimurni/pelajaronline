HTML & CSS

Pada dasarnya, CSS berfungsi untuk mendesain, membentuk, serta mengubah tampilan halaman sebuah website. CSS dapat bekerja dan berlaku melalui tag HTML. Dengan adanya CSS, tag HTML yang sederhana dapat diubah sehingga tampilan laman website pun menjadi terlihat lebih menarik dan efisien.

jQuery

Library JavaScript yang cepat,kaya akan fitur,dan mudah digunakan yang menangani animasi dan permintaan Ajax, dimana jQuery ini akan meng-compres berbagai baris atau line kode ke dalam satu buah fungsi sehingga anda tidak perlu menulis kembali semua baris kode untuk menyelesaikan satu task

Ruby

Bahasa dinamis,serba guna yang sederhana dan produktif.Ruby sering digunakan untuk membuat aplikasi web yang responsive. ruby menyediakan begitu banyak kemudahan syntax yang sederhana,dalam bahasa ruby anda tidak perlu khawatir lagi mengenai eroro karena titik koma (;) diakhir karena ruby tidak menggunakan tanda tersebut.

PHP

Bahasa pemograman skrip open source yang dapat disematkan ke dalam HTML,serta digunakan luas untuk penanganan pembuatan dan pengembangan sebuah situs web dan Website dinamis web yang kontennya bisa berubah-ubah setiap saat atau sesuai data yang tersimpan di server.juga cocok untuk pengembangan web.
